# 📊 Comparaison de l’Évolution des Prix de Deux Actions

## 📌 Objectif  
Comparer les performances de deux actions (ex: Apple vs Microsoft) sur les 3 derniers mois en utilisant Python.

## 🛠️ Technologies  
- Python (Pandas, Matplotlib, yfinance)  
- Yahoo Finance API
